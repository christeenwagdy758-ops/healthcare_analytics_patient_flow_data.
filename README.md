# Hospital Emergency Room Analysis

## Overview
Analysis of 9,216 hospital emergency room patient records to uncover patient flow patterns, wait times, satisfaction trends, and admission insights.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Power BI

## Key Insights
- August is the busiest month with 1,024 patients — double January & February
- General Practice drives the most referrals with 1,840 cases
- Average wait time is 35 minutes — consistent across all patient groups
- 50% admission rate — half of all ER patients require hospitalization
- Admitted patients report higher satisfaction (5.1 vs 4.9)
- Adults (36–60) are the largest patient group at 32% of total visits
